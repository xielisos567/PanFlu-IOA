# PanFlu-IOA
## Introduction
The PanFlu-IOA (Pan-Influenza Iteratively Optimized Antigen) toolkit was designed to overcome systemic bias and extract universal antigenic features across diverse evolutionary branches.

## Dependencies
* [Python＞=3.7](https://www.python.org/downloads/)
* Python modules: 
    * [Pandas](http://pandas.pydata.org/pandas-docs/stable/install.html)
    * [numpy](https://numpy.org/)
    
## Running
#### Running PanFlu-IOA_code1
```
python3 PanFlu-IOA_code1.py -i ./test_data/ -o ./result -t 50 -p 10
```

#### Running PanFlu-IOA_code2
```
python3  PanFlu-IOA_code2.py -i ./test_data/ -o ./result -p 10 -n 50,250
```
